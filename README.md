# 🍽️ Arisha Kitchen — Restaurant Website

A modern and responsive **restaurant website** built for **Arisha Kitchen**, a North Indian and Mughlai restaurant located in **Katwaria Sarai, New Delhi**.

The project presents the restaurant’s **menu, gallery, location, and brand identity** through a visually engaging web interface.

---

# 🚧 Project Status

**This project is currently ongoing and under active development.**

Additional improvements, features, and UI enhancements are planned for future updates.

---

# 🌐 Project Overview

This project demonstrates a **restaurant landing website** designed to provide visitors with key information about the restaurant including:

* Restaurant introduction
* Menu highlights
* Food photo gallery
* Restaurant location
* Contact section with Google Maps

The design focuses on **clean layout, elegant typography, and smooth user experience**.

---

# ✨ Features

✔ Fullscreen **hero section with background video**
✔ Smooth **CSS animations and transitions**
✔ Interactive **Back-to-Top button using JavaScript**
✔ Restaurant **menu showcase with prices**
✔ Food **photo gallery grid**
✔ Embedded **Google Maps location**
✔ Responsive layout for multiple screen sizes

---

# 🛠 Technologies Used

* **HTML5**

# 📌 Key Sections

### Hero Section

A fullscreen landing section with a **background video, restaurant title, and call-to-action button**.

### About Section

Introduces the restaurant and its cuisine style.

### Menu Section

Displays popular dishes including:

* Shahi Paneer
* Matar Paneer
* Paneer Tikka
* Chicken Handi
* Mutton Handi
* Biryani

### Gallery

A grid layout displaying food images.

### Contact Section

Contains restaurant address and **embedded Google Maps location**.
